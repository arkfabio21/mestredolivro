# Banco de Citações

Diretório para armazenar citações coletadas durante a pesquisa.

## Estrutura Recomendada

### Formato de Citação

```markdown
## "[Texto da citação]"
**Autor:** Nome Completo
**Obra:** Título do Livro/Artigo
**Ano:** 2024
**Página:** 123 (se aplicável)
**Contexto:** Para qual capítulo/tema esta citação é relevante
**Tags:** #tema1 #tema2
```

## Categorias de Citações

- `inspiracionais.md` - Citações para aberturas de capítulo
- `tecnicas.md` - Citações de especialistas/autoridades
- `historicas.md` - Citações de contexto histórico
- `estatisticas.md` - Dados e números relevantes
