# Dados e Estatísticas

Diretório para armazenar dados, estatísticas e informações factuais.

## Estrutura Recomendada

### Formato de Dados

```markdown
## Título do Dado/Estatística

**Valor:** 85% dos usuários...
**Fonte:** Nome da Pesquisa/Instituto
**Data:** 2024
**Link:** https://... (se disponível)
**Verificado:** [ ] Sim / [ ] Não
**Notas:** Observações sobre metodologia ou limitações
```

## Categorias

- `pesquisas/` - Resultados de pesquisas e estudos
- `mercado/` - Dados de mercado e tendências
- `demograficos/` - Informações demográficas
- `casos/` - Estudos de caso e exemplos reais

## Validação

Todos os dados devem ser verificados antes de usar no livro:

1. [ ] Fonte confiável identificada
2. [ ] Data de publicação verificada
3. [ ] Metodologia adequada
4. [ ] Dados reproduzíveis
