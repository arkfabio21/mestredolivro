# Conceitos

Diretório para armazenar definições e termos-chave do livro.

## Estrutura Recomendada

Cada conceito pode ser um arquivo `.md` separado ou agrupado por categoria.

### Formato de Arquivo

```markdown
# Nome do Conceito

## Definição
Explicação clara e objetiva.

## Exemplos
- Exemplo 1
- Exemplo 2

## Relações
- Relacionado a: [Outro Conceito](outro-conceito.md)

## Fontes
- Autor (Ano). Título. Editora.
```
